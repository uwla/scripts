# SCRIPTS

Shell scripts for automation.

## Description

- char: print the character matching provided positions in ASCII table
- lsort: sort lines by length
- mkcp: make a copy of the given files (suffixed with -copy)
- ord: print the position of a character in ASCII table
- repeat: run a command many times sequentially
- rmfiles: remove the files whistle keep the directories
- touchp: touch files and create directories if needed
